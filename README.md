# Gym tracker

A dead-simple and entirely vibe-coded solution for tracking your progress at the gym.

## Get started

The project is available to use on [GitHub Pages](https://finleyowen.github.io/gym-tracker/)! No setup is required. The system only ever stores your data locally (on your device).

## Note on AI usage

Note that while all the code for this project was generated by Claude Sonnet 4.6, the solution was designed by a human (me). The [design folio](./design/README.md) and the UML diagrams inside were created manually by a human (also me).

## Note on project setup

The actual source code for the project is tracked in the [gh-pages branch](https://github.com/finleyowen/gym-tracker/tree/gh-pages) of this repository. No code is stored in the main branch. This is because the project's source code consists of a single HTML file which is the file served by GitHub Pages.
